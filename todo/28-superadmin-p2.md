# 28 — SuperAdmin — Phase 2

## Current State
Tenant management, status transitions, feature toggling, platform billing, support tickets.

## Phase 2 Enhancements

### 1. Platform Analytics Dashboard
- MRR, ARR, school growth over time, churn rate
- Active users per school, feature adoption rates
- **Why:** This is how you run a SaaS business. Without platform analytics, you're flying blind

### 2. Tenant Health Score
- Composite score per school: login frequency, feature usage, fee collection rate, support ticket volume
- Auto-flag at-risk tenants (low engagement, approaching renewal)
- **Why:** Proactive retention beats reactive churn. Know which schools need attention before they cancel

### 3. Invoice Payment Tracking
- Track which platform invoices are paid/unpaid
- Automated overdue reminders to school admin
- Revenue forecasting based on current tenants and renewal dates
- **Why:** Billing without payment tracking is an accounting nightmare
