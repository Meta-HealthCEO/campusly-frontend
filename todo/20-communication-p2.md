# 20 — Communication — Phase 2

## Current State
One-way broadcast messaging (admin to parents/staff), templates, delivery tracking.

## Phase 2 Enhancements

### 1. Two-Way Messaging
- Parent can reply to messages from the school
- Teacher-parent direct messaging (scoped to their class's parents)
- Threaded conversations
- **Why:** One-way communication feels like a loudspeaker, not a relationship. Two-way messaging builds trust and resolves issues faster

### 2. Message Scheduling
- Admin composes a message and schedules it for a future date/time
- Preview scheduled messages with edit/cancel
- **Why:** Principals write newsletters on Friday for Monday delivery. Scheduling lets them batch work without worrying about timing

### 3. Read Receipts Dashboard
- Per-message: who read, who didn't, when they read
- Re-send to unread recipients with one click
- **Why:** "Did parents actually see the notice?" — now the admin knows for sure and can follow up with those who didn't
