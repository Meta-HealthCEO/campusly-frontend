# 02 — School — Phase 2

## Current State
School CRUD, multi-tenancy, subscription tiers, module feature flags, academic settings.

## Phase 2 Enhancements

### 1. School Branding
- Custom logo, primary/accent colors, school motto displayed across the platform
- Custom login page with school branding
- **Why:** Schools want to feel like the platform is "theirs", not generic SaaS

### 2. Subscription Dashboard
- Visual subscription status with renewal date, days remaining
- Usage meters (students enrolled vs. limit, modules active vs. available)
- Upgrade/downgrade self-service with tier comparison
- Expiry warning banner 30/14/7 days before renewal
- **Why:** Reduces churn. Schools that see value metrics renew. Surprise expiry = angry principals

### 3. School Year Rollover Wizard
- Guided end-of-year process: archive current year, promote students, reset attendance, carry forward balances
- Preview of what changes before committing
- **Why:** Year-end is chaos for school admins. A wizard that handles it in 5 clicks is worth the entire subscription fee
