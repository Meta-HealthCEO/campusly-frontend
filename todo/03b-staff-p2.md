# 03b — Staff — Phase 2

## Current State
Teacher creation and listing. Scoped to school. Temp password assigned.

## Phase 2 Enhancements

### 1. Staff Management CRUD
- Update and deactivate teacher accounts (currently only create + list)
- Manage subject/class assignments
- **Why:** Basic missing functionality. Admins need to edit staff records

### 2. Staff Leave Management
- Leave request submission (sick, personal, family) with date range
- Admin approval workflow
- Calendar view of staff availability
- Auto-flag classes needing substitutes
- **Why:** When a teacher is absent, the admin scrambles. A leave calendar that auto-triggers substitute assignment is transformative

### 3. Teacher Dashboard
- My classes, my subjects, upcoming assessments, pending homework to grade
- Quick actions: take attendance, post marks, create assignment
- **Why:** Teachers are the most frequent users. Their dashboard should be the most useful page in the system
