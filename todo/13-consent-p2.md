# 13 — Consent — Phase 2

## Current State
Digital consent forms, parent sign/decline with audit trail.

## Phase 2 Enhancements

### 1. Automated Consent Reminders
- SMS/push reminder for unsigned consents 3 days and 1 day before expiry
- Admin dashboard showing consent completion rates per form
- **Why:** Teachers spend hours chasing unsigned slips. Automated reminders do it for them

### 2. PDF Export of Signed Forms
- Download signed consent form as PDF with parent signature, timestamp, and IP
- Bulk download all responses for a form as ZIP
- **Why:** Schools need paper records for audits and legal compliance. Digital-only isn't enough

### 3. Consent Analytics
- Completion rate per form, average time to sign, parents who consistently delay
- **Why:** Helps admin identify which parents need a phone call and which forms need simpler language
