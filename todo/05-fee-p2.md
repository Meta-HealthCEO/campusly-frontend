# 05 — Fee — Phase 2

## Current State
Full financial lifecycle: fee types, invoices, payments, credit notes, debtors, collections, payment arrangements, statements.

## Phase 2 Enhancements

### 1. Payment Gateway Integration
- Paystack or Yoco integration for card/EFT payments
- Parent pays directly from invoice page
- Auto-reconciliation on webhook callback
- **Why:** This is the #1 feature that makes Campusly a revenue tool for schools. If parents can pay fees from their phone, collection rates skyrocket

### 2. Automated Payment Reminders
- Configurable SMS/email reminders at 7 days, 3 days, and 1 day before due date
- Overdue escalation: gentle reminder → formal notice → final demand
- Admin can preview and customize message templates
- **Why:** Schools waste hours chasing payments. Automated reminders recover fees while the admin focuses on education

### 3. PDF Statements & Receipts
- Generate and download/email PDF fee statements per student
- Payment receipts auto-generated on successful payment
- School branding on all documents
- **Why:** Parents need receipts for tax, employer claims, or their own records. PDF download is table-stakes for a financial module

### 4. Fee Analytics Dashboard
- Collection rate trends, outstanding vs. collected by month
- Debtor ageing visualization (pie chart by bucket)
- Comparison by grade/class
- **Why:** Bursars and principals need to see financial health at a glance
