# 11 — Attendance — Phase 2

## Current State
Per-period attendance tracking, discipline incidents, merits/demerits, lesson plans, substitute teachers.

## Phase 2 Enhancements

### 1. Automated Absence Notifications
- SMS/push to parent within 30 minutes of a child being marked absent
- Configurable per school (opt-in/out)
- **Why:** This is the #1 requested feature at every school. A parent should never find out at 3pm that their child wasn't at school

### 2. Chronic Absenteeism Alerts
- Auto-flag students whose attendance drops below 80% (configurable threshold)
- Dashboard widget for admin/teachers showing at-risk students
- **Why:** Early intervention prevents dropout. The system should surface this without anyone having to run a report

### 3. Attendance Analytics
- Heatmap: attendance by day of week, period, class
- Trend lines per student over the term
- School-wide attendance rate with target line
- **Why:** Principals need to see patterns. "Mondays have 15% more absences" is actionable insight

### 4. Medical Excuse Upload
- Parent uploads medical certificate when submitting an excuse
- Attached to the attendance record for audit
- **Why:** Schools need proof for excused absences. Currently this is done with paper notes that get lost
