# 04 — Parent — Phase 2

## Current State
Parent profiles, child linking, communication preferences.

## Phase 2 Enhancements

### 1. Daily Digest
- Morning push/email summary per child: today's timetable, pending homework, upcoming tests, any announcements
- Evening summary: attendance recorded, any incidents, homework submitted
- **Why:** Parents want to be involved but don't have time to log in daily. A digest keeps them connected without effort

### 2. Parent-Teacher Meeting Scheduler
- Teachers open time slots, parents book
- Calendar integration (Google Calendar link)
- Automated reminders before the meeting
- **Why:** Parent-teacher meetings are logistically painful. A booking system saves hours of admin time

### 3. Multi-Child Family View
- Single dashboard showing all children's key metrics side by side
- Quick switch between children
- Combined fee summary across siblings
- **Why:** Families with 2-3 kids in the same school shouldn't have to navigate separately for each child
