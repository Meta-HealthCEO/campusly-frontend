# 09 — Homework — Phase 2

## Current State
Assignment creation, submission with file uploads, grading with feedback, late-submission detection.

## Phase 2 Enhancements

### 1. Assignment Templates
- Teachers save and reuse assignment structures across terms
- Clone from previous year with date adjustment
- **Why:** Teachers set the same types of assignments every year. Templates save hours of repetitive setup

### 2. Deadline Reminders
- Push notification to students 24h and 2h before due date
- Parent notification for unsubmitted homework past due date
- **Why:** Late submissions drop dramatically when students get timely reminders

### 3. Parent Homework Visibility
- Parents see upcoming homework, submission status, and grades per child
- No action required — just visibility
- **Why:** "Did you do your homework?" — now parents can check instead of asking

### 4. Teacher Assignment Analytics
- Submission rate per assignment, average mark, on-time vs. late ratio
- Identify students who consistently submit late or not at all
- **Why:** Helps teachers intervene early with at-risk students
