# 12 — Achiever — Phase 2

## Current State
Achievement awards, house points, wall of fame, house leaderboard.

## Phase 2 Enhancements

### 1. Gamification Engine
- Achievement streaks: "5 days perfect attendance", "3 homework submissions in a row"
- Levels/XP system that aggregates across achievement types
- Visual badges with progress rings showing how close to next level
- **Why:** Kids are motivated by games. Turning positive behavior into a game drives engagement like nothing else

### 2. Achievement Notifications
- Push notification to student + parent when an achievement is awarded
- Weekly "Achievement Digest" to parents
- **Why:** Recognition loses impact if no one sees it. Instant notification makes the award feel real

### 3. House Competition Dashboard
- Live scoreboard visible school-wide
- Recent point awards feed (like a social timeline)
- Breakdown by category (academic, sport, cultural, behaviour)
- **Why:** House spirit drives school culture. A live dashboard makes it visible and competitive

### 4. Custom Achievement Templates
- Admin creates available badges with criteria, icons, and point values
- Teachers award from the template library
- **Why:** Standardizes recognition across the school and makes awarding faster
