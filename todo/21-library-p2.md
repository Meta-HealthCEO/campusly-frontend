# 21 — Library — Phase 2

## Current State
Book catalog, loans, overdue detection, reading challenges, reading logs, leaderboards.

## Phase 2 Enhancements

### 1. Book Reservation Queue
- When all copies are checked out, student joins a waitlist
- Auto-notification when a copy becomes available
- **Why:** Popular books are always out. A reservation queue is fair and encourages reading

### 2. Overdue Fine Integration
- Automatic fine calculation based on days overdue (configurable rate)
- Fine added to student's fee account or wallet deduction
- **Why:** Without consequences, books don't come back. Fines (even small ones) drive returns

### 3. Reading Recommendations
- "Students who read this also read..." based on school-wide reading logs
- Teacher-curated reading lists per subject/grade
- **Why:** Getting kids to read the next book is the hardest part. Recommendations make discovery easy

### 4. Reading Goal Tracker
- Student sets a personal goal (e.g., 10 books this term)
- Visual progress bar on student dashboard
- Certificate when goal is reached
- **Why:** Personal goals + visible progress = intrinsic motivation. Certificates are shareable wins
