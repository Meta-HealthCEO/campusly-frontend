# 03 — Student — Phase 2

## Current State
Student profiles, enrollment status, medical profiles, guardian linking, search by admission number.

## Phase 2 Enhancements

### 1. Bulk Enrollment (CSV Import)
- Upload CSV, map columns, preview, import
- Duplicate detection by ID number or admission number
- Error report for failed rows
- **Why:** Schools onboard 100+ students at year start. One-by-one is not an option

### 2. Student Profile Photo
- Photo upload on student profile (admin + student self-service)
- Displayed on profile cards, class lists, attendance sheets
- **Why:** Teachers need to match faces to names. Photos make every list more useful

### 3. Student 360 View
- Single page showing everything about a student: academics, attendance, fees, achievements, discipline, library, wallet — all in tab panels
- Quick-glance summary cards at top
- **Why:** When a teacher or parent asks "how is this child doing?", the answer should be one click away. This is the killer feature that makes Campusly indispensable

### 4. Academic Transcript
- Auto-generated transcript from marks data across terms/years
- Downloadable as PDF
- **Why:** Students transfer schools. Parents need transcripts for applications. Currently requires manual admin work
