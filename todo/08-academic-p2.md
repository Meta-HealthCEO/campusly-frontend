# 08 — Academic — Phase 2

## Current State
Grades, classes, subjects, timetable, assessments, marks, exams, past papers, remedial tracking, promotion, LURITS export.

## Phase 2 Enhancements

### 1. Bulk Mark Import
- Teacher uploads Excel/CSV with student marks per assessment
- Column mapping, validation, preview before commit
- **Why:** Teachers enter marks on paper or spreadsheets first. Forcing manual re-entry per student is a dealbreaker

### 2. Class Performance Analytics
- Per-subject: class average, highest, lowest, distribution histogram
- Per-student: performance trend across terms
- Comparison across classes for the same subject
- **Why:** Teachers and HODs need to spot struggling classes or topics early. Data without visualization is just noise

### 3. Automated Report Card Generation
- Pull marks per student per term, apply grading system, generate PDF report card
- School branding, teacher comments field, principal signature block
- Bulk generation for an entire class
- **Why:** Report cards are the single most admin-heavy task every term. Automating them saves days of work

### 4. Academic Calendar
- Term dates, exam periods, holidays — all in a shared calendar
- Auto-populated from school settings
- Visible to all roles
- **Why:** "When does term 2 start?" shouldn't require a phone call to the office
