# 06 — Wallet — Phase 2

## Current State
Prepaid cashless system, daily spending limits, wristband linking, transaction history.

## Phase 2 Enhancements

### 1. Auto Top-Up
- Parent sets a minimum balance threshold and top-up amount
- When balance drops below threshold, auto-charge linked payment method
- **Why:** No child should go hungry because a parent forgot to top up. Auto top-up solves this completely

### 2. Spending Insights for Parents
- Weekly spending summary: what was bought, how much, which days
- Category breakdown (food, stationery, etc.)
- Spending trends chart
- **Why:** Parents want visibility into what their child spends money on at school

### 3. Low Balance Alerts
- Push/SMS notification when wallet balance falls below configurable threshold
- **Why:** Simple but high-impact. Prevents the "I didn't know it was empty" problem
