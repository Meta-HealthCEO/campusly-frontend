# 16 — Event — Phase 2

## Current State
Event CRUD, RSVP, ticketing with QR check-in, seat reservation, photo gallery.

## Phase 2 Enhancements

### 1. Event Reminders
- Automated reminder to RSVPd attendees 1 day and 2 hours before event
- Calendar invite attachment (iCal)
- **Why:** No-shows drop when you remind people. Calendar integration makes it seamless

### 2. Waitlist
- When capacity is full, parents join a waitlist
- Auto-notify and offer ticket when a spot opens
- **Why:** Sold-out events with empty seats because someone cancelled = wasted opportunity

### 3. Event Analytics
- Attendance vs. RSVP ratio, revenue per event, no-show rate
- Year-over-year comparison for recurring events
- **Why:** Helps the school plan better events. "Our concert sells out every year, we need a bigger venue" is a data-driven decision

### 4. Ticket Refunds
- Parent requests refund, admin approves/declines
- Auto-credit to wallet or original payment method
- **Why:** Life happens. A refund flow prevents angry parent emails and admin headaches
