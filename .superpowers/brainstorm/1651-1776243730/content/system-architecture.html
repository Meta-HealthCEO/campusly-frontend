Key Design Decisions

  • API Gateway is the single entry point — handles auth (JWT + API keys), rate limiting, request routing, and tenant isolation
  • Core Platform handles all business logic — talks to the Laravel gateway via REST for payment operations
  • AI Engine is stateless — pulls data from Postgres, runs analysis, returns results. Scales horizontally
  • POS as PWA — works offline, syncs when connected. No native app needed initially
  • Redis pub/sub for real-time events between Core and AI (e.g., "new transaction" triggers fraud check)
