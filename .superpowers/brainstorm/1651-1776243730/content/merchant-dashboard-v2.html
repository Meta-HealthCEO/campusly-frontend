Design Language Applied

  • Neutral palette — charcoal primary (#171717), muted grays, no bright accent colors
  • Subtle borders — rgba(0,0,0,0.06) instead of bold lines
  • 12px rounded corners — consistent rounded-xl cards
  • Icon backgrounds — bg-primary/10 pattern from Campusly StatCards
  • Typography — tight tracking on headings, 500 weight for labels, 700 for values
  • AI Insight — understated left-border accent, not a gradient banner
  • Table — alternating subtle row backgrounds, pill-shaped status badges
  • Spacing — generous 24px padding, 16px card gaps
