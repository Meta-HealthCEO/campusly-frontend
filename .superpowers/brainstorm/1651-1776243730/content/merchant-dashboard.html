Merchant Dashboard — Key Screens

This is the main overview. The full merchant experience includes:

  • Dashboard — what you see above (revenue, transactions, AI insights, payment channels)
  • Transactions — searchable/filterable transaction history with export
  • Reconciliation — automated matching engine with manual review for exceptions
  • Analytics & BI — deep dive charts, cohort analysis, custom reports
  • AI Insights — predictive forecasts, anomaly alerts, business recommendations
  • POS — point of sale interface (separate screen coming)
  • Inventory — product catalog, stock levels, low stock alerts
  • Integrations — connect/configure Xero, Sage, ERPs
  • Settings — API keys, team management, billing, notifications
