Reconciliation Engine — How It Works

  • Auto-ingests bank statements (CSV/OFX/MT940) and matches against gateway transactions
  • 4-stage pipeline: Exact match → Fuzzy match → Manual review → Discrepancy flag
  • Per-channel breakdown shows which payment methods have matching issues
  • Bulk match for clearing multiple similar exceptions at once
  • Xero/Sage sync — matched transactions auto-post to connected accounting software
  • AI layer learns from manual matches to improve future auto-matching accuracy
