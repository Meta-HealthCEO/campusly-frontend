How should the textbook library look?

Pick the view style that feels right for a school textbook library. You can select multiple — we can combine them with a toggle.

A

Bookshelf / Bookstore View

Textbooks grouped by grade, displayed as "book covers" with spines. Each grade is a shelf row. Think Google Play Books or Apple Books. Visual, browsable, feels like a real library.

B

Compact List View

Dense table/list rows — title, subject, grade, chapters, status all visible at a glance. Efficient for admins managing many textbooks. Think spreadsheet-like.

C

Grouped Grid (current, improved)

Keep the card grid but group by grade with section headers: "Grade 12", "Grade 11", etc. Smaller cards, collapsible grade sections. More organised than the current flat grid.

D

Combo: Bookshelf default + List toggle

Bookshelf view as the default experience (visual, browsable), with a toggle icon to switch to list view when you need to manage/find things quickly. Best of both worlds.
